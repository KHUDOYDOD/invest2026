## 🎯 Что показывается:

### 1. 👥 Активных инвесторов
Количество пользователей на платформе
- Пример: `15420` → отобразится как `15.4K+`

### 2. 💰 Общие инвестиции
Общая сумма инвестиций в долларах
- Пример: `2850000` → отобразится как `$2.9M`

### 3. 📈 Средняя доходность
Средний процент доходности
- Пример: `24.8` → отобразится как `24.8%`

### 4. 🏆 Надежность
Процент надежности платформы
- Пример: `99.9` → отобразится как `99.9%`

- Рекомендуется: 20-30%
- Слишком высокая (>50%) выглядит подозрительно

## 🔧 ТЕХНИЧЕСКИЕ ДЕТАЛИ:

### База данных:
Таблица: `platform_statistics`
- `users_count` - количество инвесторов
- `investments_amount` - сумма инвестиций
- `profitability_rate` - доходность
- `reliability` - надежность

### API:
- `GET /api/statistics` - получить данные
- `PUT /api/statistics` - обновить данные

### Страницы:
- Главная: `http://localhost:3000`
- Админка: `http://localhost:3000/admin/statistics`

## 🆘 РЕШЕНИЕ ПРОБЛЕМ:

### Проблема: Страница не открывается
**Решение:** Убедитесь что сервер запущен
```bash
npm run dev
```

### Проблема: Изменения не сохраняются
**Решение:** Проверьте что вы вошли как админ

### Проблема: Данные не отображаются
**Решение:** Обновите страницу (F5)

### Проблема: Ошибка базы данных
**Решение:** Запустите скрипт
```bash
node add-reliability-field.js
```
