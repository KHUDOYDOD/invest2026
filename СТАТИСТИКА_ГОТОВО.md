## Структура данных:

### Таблица platform_statistics:
```sql
- users_count (INTEGER) - Количество активных инвесторов
- users_change (DECIMAL) - Изменение количества инвесторов в %
- investments_amount (BIGINT) - Сумма месячных инвестиций в $
- investments_change (DECIMAL) - Изменение инвестиций в %
- payouts_amount (BIGINT) - Сумма выплаченной прибыли в $
- payouts_change (DECIMAL) - Изменение выплат в %
- profitability_rate (DECIMAL) - Средняя доходность в %
- profitability_change (DECIMAL) - Изменение доходности в %
- updated_at (TIMESTAMP) - Время последнего обновления
```

## Как использовать:

### Для админа:
1. Зайдите в админ панель: http://localhost:3000/admin
2. В меню "Контент" выберите "Статистика сайта"
3. Измените нужные цифры в форме
4. Нажмите "Показать превью" чтобы увидеть как будет выглядеть
5. Нажмите "Сохранить изменения"
6. Изменения сразу отобразятся на главной странице

### Примеры значений:
- **Активные инвесторы**: 15420 (отобразится как 15.4K)
- **Изменение**: 12.5 (отобразится как +12.5%)
- **Месячные инвестиции**: 2850000 (отобразится как $2.9M)
- **Выплачено прибыли**: 1920000 (отобразится как $1.9M)
- **Средняя доходность**: 24.8 (отобразится как 24.8%)

### Советы:
- Используйте реалистичные цифры для доверия пользователей
- Изменения могут быть положительными (+) или отрицательными (-)
- Большие числа автоматически форматируются (K для тысяч, M для миллионов)
- Изменения сразу видны на главной странице после сохранения
