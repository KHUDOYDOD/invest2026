📝 ДЛЯ РАЗРАБОТЧИКОВ
═══════════════════════════════════════════════════════════════

Как добавить переводы в новую страницу:

1. Импортируйте хук:
   import { useLanguage } from '@/contexts/language-context'

2. Используйте в компоненте:
   const { t } = useLanguage()

3. Замените тексты:
   <h1>{t('dashboard.title')}</h1>

Все ключи уже есть в файлах:
lib/i18n/locales/*.json
