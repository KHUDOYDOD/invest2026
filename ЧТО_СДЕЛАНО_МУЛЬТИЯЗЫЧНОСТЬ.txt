🚀 КАК ЗАПУСТИТЬ И ПРОВЕРИТЬ
═══════════════════════════════════════════════════════════════

1. Запустите сервер:
   npm run dev

2. Откройте сайт:
   http://localhost:3000

3. Найдите переключатель языков в Header (справа вверху)

4. Нажмите на кнопку с глобусом 🌐

5. Выберите любой язык

6. Проверьте, что Header переведен!
