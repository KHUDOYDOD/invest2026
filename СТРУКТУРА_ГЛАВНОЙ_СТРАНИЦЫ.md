## 📊 Какие секции можно редактировать через админ панель:

### ✅ 1. STATISTICS (Статистика) - `/admin/statistics`
**Редактируется:** ДА ✅
- Активные инвесторы (число + изменение %)
- Месячные инвестиции (сумма + изменение %)
- Выплачено прибыли (сумма + изменение %)
- Средняя доходность (процент + изменение %)

### ✅ 2. INVESTMENT PLANS (Тарифы) - `/admin/investment-plans`
**Редактируется:** ДА ✅
- Названия планов
- Проценты доходности
- Минимальные суммы
- Описания

### ✅ 3. TESTIMONIALS (Отзывы) - `/admin/testimonials`
**Редактируется:** ДА ✅
- Модерация отзывов
- Одобрение/отклонение
- Просмотр всех отзывов

### ⚠️ 4. HERO SECTION (Главный баннер) - `/admin/hero-settings`
**Редактируется:** ЧАСТИЧНО ⚠️
- Заголовок
- Описание
- Кнопки

### ⚠️ 5. NEWS SECTION (Новости) - `/admin/news`
**Редактируется:** ЧАСТИЧНО ⚠️
- Добавление новостей
- Редактирование новостей

### ❌ 6. Остальные секции
**Редактируется:** НЕТ ❌
- User Activity Rows (из базы данных автоматически)
- New Users Showcase (из базы данных автоматически)
- Live Activity Feed (из базы данных автоматически)
- Features (статичный контент)
- Mobile App (статичный контент)
- Trust Indicators (статичный контент)

## 🔧 Как добавить новую секцию:

1. Создайте компонент в `components/`
2. Импортируйте в `app/page.tsx`
3. Добавьте в нужное место на странице
4. При необходимости создайте API в `app/api/`
5. Создайте админ страницу в `app/admin/`
